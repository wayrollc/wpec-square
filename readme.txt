== Description ==

This plugin will add the Square gateway to your list of payment gateways in WP eCommerce. 


== Installation ==

1.). Upload the folder 'wpec-square' to the '/wp-content/plugins/' directory or use the plugin up loader in the WordPress dashboard. ( Plugins -> add new -> upload ) 

2.). Activate the WP eCommerce Square plugin from the 'Plugins' menu in WordPress.

3.) The Square Gateway will appear in your list of payment gateways in Settings -> Store -> Payments.

4.) Put your Square API info in the required fields
